** README **

On successfull form submission a mail with the pdf link will be sent to both admin and the user and the attachment will be stored inside the "masterpdf" folder located under the UPLOADS folder.

== Configuration ==

*Once the plugin is installed and activated add the shortcode "[master-form]" to the required form page.
*Add a folder with name "masterpdf" under the UPLOADS folder. https://www.screencast.com/t/KUlE1Yfmi1P
